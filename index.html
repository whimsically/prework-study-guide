HTML

  • The head element contains information about the webpage:
    • The title of the webpage, in a title tag
    • Any meta information, such as viewport and charset
    • Link to external stylesheet
  • The body element represents the visible content shown to the user.
  • Accessibility is important:
    • Only use one h1 tag
    • Other header tags should be in order
    • Include alt text on images

CSS

  • CSS adds the styling, such as colors and layout.
  • CSS stands for Cascading Style Sheets
  • CSS can be inline, put in a style tag in the header, or linked to an external file. The latter is preferred for flexibility
  • With CSS, you select an element and then apply styles to it.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • cd: shows current directory
  • ls shows all files in the current directory

Javascript

  • Javascript adds the functionality to a page
  • A variable is a named container that allows us to store data in our code.
  • An "if" statement allows us to use conditional logic
  • An array is used to store a group of related data
  • console.log prints to the console
  • Control flow is the order in which a computer executes code in a script.
